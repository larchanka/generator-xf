# Yeoman Generator for [X-Framework](https://github.com/epam/x-framework)


## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator: `npm install -g generator-xf`
- Run: `yo xf`
